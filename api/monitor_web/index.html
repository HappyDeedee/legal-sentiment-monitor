账号登录

登录方式按平台统一配置，任务只选择平台和关键词。推荐使用“浏览器 Profile / 扫码”，服务器无界面或排障时可切换到 Cookie；手机号登录依赖 MediaCrawler 当前平台实现，适合本地人工处理。

平台登录方式配置

操作顺序

  1. 先为每个平台选择登录方式；不确定时保留默认的“浏览器 Profile / 扫码”。
  2. 分别点击抖音、快手、小红书的“打开登录窗口”。
  3. 在弹出的浏览器里完成扫码、手机号验证或平台要求的安全验证。
  4. 确认网页已经是登录状态后，关闭这个登录窗口。
  5. 回到后台点击“刷新登录状态”，再去任务管理里立即运行。
说明：Cookie 会加密保存，页面只显示是否已保存；快手当前 MediaCrawler 代码未实现手机号登录，因此后台不开放快手手机号模式。
