采集规则说明
当前首版以平台搜索词采集为主。平台搜索词会用于平台搜索；律所名称、别名和排除词用于评估、报告和采集后过滤。

新建任务

任务配置负责串联调度、平台采集、AI 评估和邮件发送；账号、代理、AI 接入、邮件模板在各自资源页独立维护。

基本信息

律所名称和别名用于 AI 判断、报告标题和线索归属，不会自动追加为平台搜索词。

采集设置

平台搜索词会逐行传入平台搜索。账号和代理只在这里绑定，资源资料仍在资源管理中维护。

这里的每一行才会用于平台搜索；律所别名不会自动参与搜索。
采集参数
当前任务按平台搜索词运行。平台搜索词、平台和采集范围会共同决定本次采集内容。

过滤与去重

排除词不参与平台搜索，只在内容采回后过滤标题、正文、作者和来源搜索词。系统最终按发布时间和内容 ID 去重。

任务调度

设置任务是否启用、多久运行一次，以及每次只处理哪个时间范围内的新内容。

AI 评估

选择用于本任务的 AI 接入资源。未选择或接入异常时，采集仍会运行，内容进入待人工复核。

邮件报告

设置报告收件人和本任务使用的邮件模板。未配置邮件也可以采集和生成报告，只是不发送邮件。
