账号登录

后台采用浏览器 Profile 登录：每个平台先打开一次登录窗口完成扫码或手机号验证，关闭窗口后，定时采集会复用该平台的登录态。任务运行时不再反复选择 qrcode、phone 或 cookie。

操作顺序

  1. 分别点击抖音、快手、小红书的“打开登录窗口”。
  2. 在弹出的浏览器里完成扫码、手机号验证或平台要求的安全验证。
  3. 确认网页已经是登录状态后，关闭这个登录窗口。
  4. 回到后台点击“刷新登录状态”，再去任务管理里立即运行。
说明：如果平台提示登录失效，重新打开对应平台登录窗口即可。Cookie、二维码、手机号是 MediaCrawler 底层登录方式，第一版后台统一用 Profile 保存登录态。
